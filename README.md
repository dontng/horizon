# 考研 200 天作战计划（华中师大 · 电子信息专硕）

> 这不是一份用来"第一天热血"的计划。它的唯一价值，是你**每周日**回来翻一次，勾掉进度、照出偷懒。
> 看不下去全部时，至少读完"北极星"和"唯一胜负手"两节，重新对准方向就够了。

## 北极星

- **目标院校 / 专业**：华中师范大学 · **085400 电子信息（专硕）**
  - 锁死专硕，**不是**学硕（081200）。学硕考英语一+数学一，你现有的英二/数二积累会全部贬值。
  - 同等学力（专科满两年）可报，代价是复试加试两门本科主干课（大概率数据结构 + 操作系统，与 408 重叠）。
  - ⚠️ 今年 9–10 月新招生简章出来后，**重新核一遍考试科目**，这两年改考频繁。
- **分数目标**：下限 **355–365**（清线 + 安全垫），冲刺天花板 **385+**。
  - 参考：该专业拟录取均分 340+，推免占走不少名额，统考竞争激烈。
  - 往高处够是为了把下限拉上来。**计划保下限、够上限。**
- **考试时间**：约 2026 年 12 月第三个周末（以官方公布为准）。

## 唯一胜负手

> **三战从未把 408 大题完整做完过一次。** 这不是能力问题，是从没进入过那个阶段。

整盘棋的成败几乎全押在一件事上：**9–10 月能不能第一次把 408 大题完整啃下来一轮。**
- 啃下来 → 上岸。
- 卡在选择题刷题的舒适区里 → 385 是空话。

**最高优先级时间（周日整块 + 状态最好的工作日晚上）永远优先喂给 408 和数学大题，其余全是配菜。**

## 时间档位规则（计划的操作系统）

时间不是均质的。按科目硬排表必崩，按"脑力档位"匹配场景才活得下去。

| 档位 | 场景 | 放什么任务 |
|---|---|---|
| **高档**（深度、需连续大脑） | 周日整块 + 一个最佳工作日晚上 | 数学二大题、408 大题、408 综合应用题 |
| **中档**（工作日晚上 2–2.5h 有效） | 洗澡煮饭之后那块 | 408 章节选择刷题、数学计算训练、英语精读一篇 |
| **低档**（碎片、可被打断） | 公司摸鱼（git pull）、通勤 | 墨墨单词、田静每日一句听写跟读、政治选择刷题、408 概念回顾 |

铁律：
- **绝不**拿摸鱼时间做数学/408 大题——做不动，只会挫败。
- 周日是最大的未开发资源，但**别第一天就上 8 小时**。从上午 3 小时雷打不动起步，2–3 周后再加下午。"周日上午必学，下午看状态"比硬扛全天可持续。
- 周六单加后那段时间 = 一个普通工作日晚上（中档任务），不用给它仪式感。

## 五个阶段

### 阶段 0（6.1 – 6.30）习惯固化 + 硬核启动
- **英语**：不加量，守住每日一句习惯（Stage 1 本来就到 6.30）。
- **408**：从最难两块先啃 —— 数据结构 → 组成原理，配章节选择。
- **数学二**：唤醒（不是重建，84 底子还在），强化阶段高数为主。
- **政治**：❌ 一个字都别碰，现在投进去会蒸发。
- **加班**：50h/月封顶，本月就开始护。

### 阶段 1（7.1 – 8.31）基础攻坚 · 全程最重的两个月
- **408**：四科基础全过 + 章节选择刷完（数据结构→组成→操作系统→网络）。
- **数学二**：强化全推，高数+线代，狂练计算。
- **英语**：Stage 2 精读，剑雅阅读当训练材料（考研真题已背熟，不能当生题）。
- **政治**：8 月底可让马原先起步（最需理解，吃早动红利）。

### 阶段 2（9.1 – 10.31）真题与大题攻坚 · 决定胜负
- **408**：⭐ **正式上大题**，先按专题、再按年份。这是三战从未到达的地方，必须在这两个月真正完成一轮。
- **数学二**：上真题、按年份限时。
- **英语**：Stage 3 真题精做、作文模板搭建。
- **政治**：选择题开刷（1000 题），马原理解、史纲毛中特记忆。

### 阶段 3（11.1 – 11.30）套卷模拟
- **408 / 数学二**：全程限时套卷。
- **政治**：肖八选择 + 大题框架。⭐ **专治写不完**：不是再背框架，是**限时写**——反复掐表写 2–3 篇大题，把手速练出来。
- **英语**：作文定稿 + 全套限时。

### 阶段 4（12.1 – 考试，约 3 周）全真模拟 + 背诵冲刺
- **周日按真实考试时间表做全真套卷**（让周日变成考场节奏预演）。
- **政治**：肖四背大题——政治大题的分主要在这一个月拿，叠加 11 月练出的手速 = 30→70。
- **英语**：作文背诵内化。
- **数学/408**：回错题、查漏。

## 失败模式备忘（每月初读一遍）

这些是你**亲手验证过**的失败方式，写在这里不是为了自责，是当护栏：

1. **工作打断学习** → 不是能力问题，是结构问题。加班 50h/月封顶，周日整块神圣不可侵犯。
2. **408 大题三战从没做完一次** → 9–10 月不进入大题阶段 = 重蹈覆辙。这是头号红线。
3. **政治大题写不完，两三篇后超时** → 框架没问题，问题是手速。11 月必须限时实写练出来。
4. **周日因为累一直休息** → 最大资源被浪费。先保上午，别贪全天。
5. **躲在选择题刷题的舒适区** → 感觉很努力，但不碰大题就是原地踏步。
